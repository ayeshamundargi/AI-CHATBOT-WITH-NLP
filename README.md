##This project is a simple NLP-based chatbot developed using Python.
It processes user input by converting text to lowercase, removing punctuation, and eliminating stopwords.
The chatbot uses TF-IDF (Term Frequency–Inverse Document Frequency) to transform text into numerical vectors.
Cosine similarity is then applied to compare user input with predefined patterns and find the best match.
It supports multiple intents like greetings, general queries, and Python-related questions.
If no suitable match is found, a fallback response ensures smooth and continuous interaction.
